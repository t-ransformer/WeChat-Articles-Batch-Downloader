# 测试数据说明

## test_articles.xlsx

这是一个用于CI/CD测试的Excel文件，包含2-3个公开可访问的微信公众号文章URL。

### 文件格式

- **列名**: `标题`、`链接`
- **数据**: 每行一个文章，包含标题和URL

### 使用要求

1. **URL要求**: 
   - 必须是公开可访问的微信公众号文章URL
   - URL格式: `https://mp.weixin.qq.com/s/xxxxx`
   - 确保URL长期稳定，不会很快失效

2. **测试数据**:
   - 建议使用2-3个URL即可（避免CI运行时间过长）
   - 选择包含图片的文章，以便测试图片下载功能
   - 选择包含LaTeX公式的文章（如果有），以便测试公式转换功能

### 更新测试数据

如果需要更新测试数据，请：

1. 打开 `test_articles.xlsx`
2. 替换为新的公开可访问的URL
3. 确保URL格式正确
4. 提交到仓库

### 注意事项

- 不要使用包含敏感信息的文章URL
- 不要使用需要登录才能访问的文章URL
- 确保URL是公开的、稳定的
